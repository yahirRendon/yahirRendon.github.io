Minesweeper is a single player puzzle game created by Curt Johnson for IBMâ€™s OS/2. The game was later ported to Windows by Curt Johnson and Robert Donner while developers at Microsoft. The game original consisted of various difficulties corresponding to the dimensions of the grid that made up the play area. Mines are randomly hidden within the individual cells that make up the grid. The objective the game is for the user to clear the cells that do not contain mines using the information provided by cleared cells through logical deductions.

When a game is initialized all the cells start in the closed state. The user can then select any cell to open or reveal it. Once opened the cell either contain a mine or it does not. If the user opens a cell that contains a mine the game is over. However, if the cell does not contain a mine the cell will have a number that represents the number of mines located within the cells that surround it. For example, a cell with the number 1 will have exactly one neighboring cell with one mine while a cell with the number 3 will have exactly 3 neighboring cells with mines. If a cell has no neighboring mines an area of the board will be cleared in all direction until neighboring cells with neighboring mines are reached. The game is won when all cells that do not contain mines are opened leaving only the cells with mines remaining.
