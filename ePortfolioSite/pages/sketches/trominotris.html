Trominotris
A minimal version of Tetris using trominoes
↺


View Code on GitHub

Instructions
SPACE KEY to play and pause game
LEFT KEY to move tromino left
RIGHT KEY to move tromino right
DOWN KEY to move tromino down
UP KEY to rotate tromino
Reset button to restart game
Discover

In 1984 Alexy Pajitnov, a Soviet software engineer, created the popular game of Tetris. The name came from combining the Greek word tetra, meaning four, and Pajitnovâ€™s favorite sport of tennis. The game consists of tetrominoes, which are geometric shapes composed of for squares and connect along an edge. The goal of the game is to clear rows created by stacking the tetrominoes as they drop. The user can move the blocks left, right, down, and rotate them but cannot move them up. As rows are cleared points are given and the rate at which the trominoes drop increases. The biggest points awarded are for clearing four rows at once otherwise called a Tetris.

As a fun programming challenge, I recreated the core mechanics of the game in a simple layout. Instead of the classic tetromino shapes my version uses trominoes, or shapes created by using three squares. Unfortunately, there are only twp unique trominoes that can be created a long piece and an L piece. So, to make the game more interesting two irregular trominoes are created that include one or more squares aligned along a vertex rather than an edge. The fall at sparse intervals but can make the game more challenging as the drop speed picks up.

Enjoy!
